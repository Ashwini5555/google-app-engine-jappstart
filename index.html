jappstart by tleese22

Java framework for Google App Engine

jappstart is a Java framework for Google App Engine built on Spring, Spring Security, and Sitemesh. The goal is to create a web framework that provides generic functionality that is common among most sites and built for Google App Engine. The project aims to be the starting point for GAE Java applications.
